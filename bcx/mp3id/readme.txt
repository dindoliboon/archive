////////////////////////////////////////////////////////////////////////////

  Get MP3v1 Tag 1.1

  Reads ID3_V1 tag from an MP3 file.

  Usage:
    MP3ID <file.mp3>
    file.mp3  name of file to examine

  Examples:
    MP3ID cool.mp3
    MP3ID "my song.mp3"
