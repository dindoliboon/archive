D-Color 1.0
~~~~~~~~~~~
D-Color is a tiny & free Windows utility that allows you to
change the appearance of the desktop icon text foreground
and background color, allowing you to show your wallpaper
as it was meant to be! D-Color was tested on both Windows
2000 Professional & Windows Millennium Edition.

Installation
~~~~~~~~~~~~
Run the file setup.exe. It will launch the NSIS installer.
