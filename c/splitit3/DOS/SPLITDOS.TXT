บ   Splitit Offers Both interactive and command line usage modes.              บ
บ   The commandline mode which is covered below allows Splitit to be           บ
บ   called from a batch file and perform its handiwork non-interactively.      บ
บ                                                                              บ
บ   Commandline:  SPLITIT [command] ROOTNAME.EXT [checksum-bytes]              บ
บ   B  Behead     remove [checksum] bytes from the beginning of the file.      บ
บ                 2-files are created... [ROOTNAME].001 and [ROOTNAME].002.    บ
บ   T  Trim       remove [checksum] bytes from the end of the file.            บ
บ                 2-files are created... [ROOTNAME].001 and [ROOTNAME].002.    บ
บ   C  Chunk      break file into pieces (multiple files) of [checksum] bytes  บ
บ      BINARY     starting with  file [ROOTNAME].001, [ROOTNAME].002, etc.     บ
บ   A  Chunk      break a file into pieces of [checksum] bytes same as binary  บ
บ      ASCII      but do not split lines of text. Use for BBS Logfiles, etc.   บ
บ   J  Join       Join a file that was previously split into pieces            บ
บ                 (multiple files) starting with  file [ROOTNAME].001.         บ
บ                 The [checksum] is not required for this option.              บ
